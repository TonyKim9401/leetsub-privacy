LeetSub 개인정보 처리방침

최종 업데이트: 2025년 11월 14일

1. 수집하는 정보

LeetSub은 다음 정보를 수집합니다:

  • LeetCode 문제 제목
  • 문제 난이도
  • 문제 설명
  • 선택한 프로그래밍 언어

개인 식별 정보는 수집하지 않습니다.

2. 정보 사용 목적

수집된 정보는 다음 용도로만 사용됩니다:

  • AI 기반 맞춤형 힌트 생성
  • 예시 정답 코드 생성

3. 정보 저장

  • 힌트와 진행 상황은 브라우저 로컬 스토리지에 저장됩니다
  • 서버에는 저장되지 않습니다
  • 사용자가 직접 삭제 가능합니다

4. 제3자 서비스

이 Extension은 다음 서비스를 사용합니다:

  • Vercel (서버 호스팅)
  • Anthropic Claude API (AI 힌트 생성)

5. 보안

모든 데이터는 HTTPS로 암호화 전송됩니다.

6. 사용자 권리

사용자는 언제든지:

  • Extension을 삭제할 수 있습니다
  • 로컬 데이터를 초기화할 수 있습니다
